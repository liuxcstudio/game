# 打工生活模拟器 - 优化版 v2.2.0

## 启动方法

### 方法一：一键启动（推荐）
```bash
node start.js
```
启动后会自动打开浏览器，默认地址：http://localhost:8080

### 方法二：指定端口
```bash
node start.js 3000
```

### 方法三：使用其他HTTP服务器
```bash
python3 -m http.server 8080
npx http-server -p 8080
```

## 优化内容

### 启动性能
- 关键脚本预加载（preload）
- 启动进度细粒度显示（4阶段提示）
- 启动耗时统计

### 渲染性能
- 后台标签页自动暂停游戏（省CPU/电）
- 自动垃圾回收（30秒周期）
- 帧率监控面板

### 服务器优化
- gzip 压缩传输（减少约70%体积）
- 静态资源强缓存（1年过期）
- 大文件流式传输（不占满内存）
- Range 请求支持（断点续传）
- 局域网访问支持

### 功能增强
- 全屏模式按钮
- 存档管理面板（导出/导入/备份/恢复/清除）
- 性能监控面板（FPS + 内存）

### 网络优化
- 统一XHR拦截器（WeakSet替代属性标记）
- 移除冗余原型覆盖
- 更精确的域名拦截列表

## 系统要求
- Node.js 12+ 或任意HTTP服务器
- 现代浏览器（Chrome 80+, Firefox 75+, Edge 80+）
- WebGL 支持
